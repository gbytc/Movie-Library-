# 🎬 Movie Library - Личная кинотека

**Автор: Керн Семен**

## 📖 Описание

Movie Library — это графическое приложение для управления личной коллекцией фильмов.  
Оно позволяет добавлять, удалять и фильтровать фильмы, а также сохраняет данные в JSON файл.

## ✨ Возможности

- ➕ Добавление фильмов с проверкой данных (год, рейтинг)
- 🗑 Удаление выбранных фильмов
- 🔍 Фильтрация по жанру и году выпуска
- 💾 Автоматическое сохранение в JSON
- 📊 Удобный табличный интерфейс

## 🚀 Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `movie_library.py`
3. Запустите командой:
   ```bash
   python movie_library.py
